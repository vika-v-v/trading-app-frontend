# Get Started
1) Install node.js (google)
2) Run "npm install -g @angular/cli"
3) Go to the Project root
4) Run "cd wertpapieranlagen"
5) Run "npm i"
6) Run "ng s"

-- new teminal --
4) Run "cd wertpapieranlagen"
4) Run "json-server --watch src/db.json"

-- new terminal --
4) Run "cd wertpapieranlagen"
6) Run "node src/server.js"
